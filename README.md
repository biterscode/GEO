# GEO: Generative Engine Optimization — Audit Hub

This repository contains comprehensive **Generative Engine Optimization (GEO)** audits and implementation plans for e-commerce brands. These reports analyze brand visibility across AI platforms (ChatGPT, Perplexity, Google Gemini, Bing Copilot) and provide technical and content-based strategies to improve citation rates.
