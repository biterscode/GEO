| No FAQ schema | Medium | /pages/faqs |
| No Article schema with dates | Medium | Blog posts |

### Schema to implement immediately

#### Homepage — Organization + WebSite (add to `theme.liquid`)
```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Happimess",
  "url": "https://happimess.com",
  "logo": "https://happimess.com/cdn/shop/files/happimess-logo.png",
  "description": "Modern storage, organization, and home management products.",
  "contactPoint": {
    "@type": "ContactPoint",
    "telephone": "+1-917-261-4961",
    "email": "hello@happimess.com",
    "contactType": "customer service"
  },
  "sameAs": [
    "https://www.instagram.com/happimess/",
    "https://www.facebook.com/happimess/",
    "https://www.tiktok.com/@happimess",
    "https://www.youtube.com/@happimess",
    "https://www.pinterest.com/happimess/"
  ]
}
```

#### WebSite with Sitelinks Searchbox (add to `theme.liquid`)
```json
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "name": "Happimess",
  "url": "https://happimess.com",
  "potentialAction": {
    "@type": "SearchAction",
    "target": "https://happimess.com/search?q={search_term_string}",
    "query-input": "required name=search_term_string"
  }
}
```

#### Product pages — fix `brand.name` (in `product.liquid`)
```json
"brand": {
  "@type": "Brand",
  "name": "Happimess"
}
```
Replace: `"name": "Happimess Dev"` → `"name": "Happimess"`

#### FAQ page schema (`/pages/faqs`)
